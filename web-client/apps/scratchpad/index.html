About Pengines ScratchpadVersion 0.9.3  

Pengines Scratchpad is part of the Pengines platform. This is where you can learn how to program the interaction between JavaScript and Prolog, and also quickly prototype your applications.

Motivation

Pengines Scratchpad is inspired by web-based tools such as JSFIDDLE in that it allows you to develop a program and then share it with others on the Web. However, lacking both debugger and profiler, Pengines Scratchpad is far from a full-fledged IDE for Prolog embedded in HTML, and to avoid compromising the server it does not allow you to execute "dangerous" Prolog predicates such as predicates reading and writing files. (It does however allow you to use pengine_input/1 and pengine_output/1 but they will only "read" from and "write" to your web browser.)

Time limit

Prolog goals will be allowed to execute on the server for 300 seconds before aborting and returning with a message stating that the time limit has been exceeded.

Known issues

  • Pengines Scratchpad works fine in Safari, Firefox and Chrome. It seems there are some problems with Internet Explorer.

Technology

Pengines Scratchpad is implemented in a combination of Prolog, HTML, CSS and JavaScript. The only framework used is Bootstrap and the choice of editor landed on the very capable ACE editor. The server serves static HTML, CSS and JavaScript code only, none of that is dynamically generated on the server. Instead, to create behaviour, the main traffic between server and client consists of the JSON responses from the server guiding the transformation of the browser's DOM on the client side.

Source code

The source code for Pengines Scratchpad is available in the Pengines distribution on GitHub.
